# Dev

1. Clonar el proyecto
2. Copiar el ```env.template``` y renonbrar a ```.env```
3. Ejecutar 
```
yarn install
```
4. Levantar la BD
```
docker-compose up -d
```
5. Levantar el backend
```
yarn start:dev
```
6. Ingresar a
```
localhost:3000/graphql
```
